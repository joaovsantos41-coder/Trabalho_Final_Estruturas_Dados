Relatório

Neste trabalho desenvolvemos três programas em linguagem C utilizando as estruturas de dados estudadas na disciplina: lista simples, pilha simples e fila simples/circular. Todos os programas utilizam struct, vetores de estruturas, menu interativo e armazenamento dos dados em arquivos CSV.

Trabalho A – Playlist de Músicas
Escolhemos o tema Playlist de Músicas por ser um exemplo simples e fácil de relacionar com o dia a dia. O programa permite cadastrar, buscar, editar, excluir e listar músicas, garantindo que cada uma possua um ID único. Os dados são salvos em arquivo CSV para que permaneçam armazenados após o encerramento do programa.

Trabalho B – Chamadas de Emergência
Escolhemos esse tema porque representa bem o funcionamento de uma pilha (LIFO), em que a última chamada registrada é a primeira a ser atendida. O sistema permite registrar chamadas, atender a última chamada e consultar o topo da pilha, além de salvar e carregar os dados em CSV.

Trabalho C – Fila de Impressão
O tema foi escolhido por representar uma aplicação comum das filas. O programa utiliza uma fila simples para trabalhos normais e uma fila circular para trabalhos prioritários, garantindo que os prioritários sejam processados primeiro. Também permite inserir, listar e processar trabalhos, utilizando arquivos CSV para armazenar as informações.

Dificuldades e decisões de implementação:
Durante o desenvolvimento, a principal dificuldade foi implementar corretamente a fila circular e a manipulação dos arquivos CSV, além de realizar as validações necessárias, como impedir identificadores duplicados e tratar estruturas vazias ou cheias. Essas dificuldades foram resolvidas por meio de testes e ajustes no código, garantindo o funcionamento correto de todas as funcionalidades.

Como decisão de implementação, optamos por utilizar vetores de struct em todos os programas, conforme solicitado no enunciado, além de menus interativos para facilitar a utilização. Também utilizamos arquivos CSV para armazenar os dados, permitindo que as informações fossem preservadas entre as execuções dos programas.

Concluindo
O desenvolvimento do trabalho permitiu aplicar na prática os conceitos de listas, pilhas e filas, além de reforçar os conhecimentos sobre manipulação de arquivos, organização de dados em estruturas e programação em linguagem C. Apesar de alguns desafios durante a implementação, principalmente no uso dos arquivos CSV e da fila circular, os objetivos propostos foram alcançados com sucesso.
